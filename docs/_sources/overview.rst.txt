Overview
========

Library for `desktop notifications <https://people.gnome.org/~mccann/docs/notification-spec/notification-spec-latest.html>`_.


Installation
------------

.. highlight:: none

::

# pip install kitsiso


Dependencies
------------

- `jeepney == 0.4 <https://pypi.org/project/jeepney/>`_
